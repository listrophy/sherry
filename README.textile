h1. Sherry?

The proper response to this is always "Rather." Therefore, this gem, when loaded, attaches the method @sherry?@ to all objects. It will always return the string "Rather."

h2. Install

bc. gem install sherry

or in your Gemfile:

bc. gem 'sherry'
